# EcoLink

O **EcoLink** é uma plataforma digital educativa projetada para auxiliar visitantes e pesquisadores durante a exploração da Caverna do Mandacaru, localizada em Campo Formoso, Bahia. O objetivo do projeto é organizar e facilitar o acesso a informações geológicas e ambientais coletadas no local.

## Funcionamento do Site

O site funciona como um guia interativo que transforma o percurso físico da caverna em uma experiência informativa digital. Ele permite que o usuário acesse conteúdos específicos sobre diferentes pontos da caverna de duas maneiras principais:

### 1. Acesso via QR Code

Ao longo da caverna, pontos de interesse possuem placas com QR Codes. Ao utilizar a função "Ler QR-Codes" no site, a câmera do celular é ativada. Assim que o código é reconhecido, o sistema redireciona o visitante automaticamente para a página de detalhes.

* **Simplicidade na criação:** Para que um novo ponto de interesse funcione, o QR Code precisa ser gerado contendo apenas o "código" (nome do arquivo) correspondente. Não é necessário criar links complexos para cada placa; o sistema do site se encarrega de encontrar o conteúdo baseado apenas no texto lido pelo QR Code.

### 2. Acesso via Código Manual

Pensando em situações onde a câmera pode ter dificuldade de foco devido à iluminação, o site oferece um campo de entrada manual.

* **O que é o código:** O código que o usuário deve digitar é exatamente o nome do arquivo de conteúdo armazenado no sistema.
* **Exemplo:** Se o visitante deseja ver informações sobre o "Salão da Cortina", basta digitar o código correspondente no campo de busca (nesse caso `cortina`). O site localizará o arquivo de texto vinculado e carregará todas as informações na tela.

## Conteúdo das Páginas

Ao acessar um ponto da caverna pelo site, o visitante encontrará:

* **Dados Ambientais:** Informações precisas sobre a umidade e a temperatura registradas naquele setor.
* **Medições:** Detalhes sobre a altura do teto e a distância entre as paredes (largura).
* **Descrições Geológicas:** Textos explicativos sobre as formações presentes (estalactites, cortinas, cones, etc.) e como elas se formaram ao longo do tempo.
* **Galeria de Fotos:** Imagens reais do local para auxiliar na identificação visual das estruturas mencionadas.

## Sobre o Projeto

O EcoLink busca democratizar o conhecimento científico. Muitas vezes, detalhes importantes sobre a preservação e a formação das cavernas ficam restritos a relatórios técnicos. Este site traz essas informações para uma linguagem simples e acessível, promovendo a educação ambiental e a conscientização sobre a importância de preservar o patrimônio natural da região de Campo Formoso.
